# 📊 Radha Stores Excel Sales Dashboard

## Project Overview

This project is an interactive Excel Dashboard built to analyze the annual sales performance of Radha Stores in 2022.

The dashboard provides insights into sales trends, customer demographics, order status, regional performance, and sales channels using Excel's data analysis and visualization features.

## Business Problem

The management team wanted a single dashboard to monitor:

- Sales performance
- Customer behavior
- Product category performance
- Sales channels
- State-wise revenue contribution

## Key Insights

### Customer Insights

- Women contribute 64% of total sales.
- Adult customers are the largest buyer segment.

### Sales Insights

- Maharashtra generates the highest revenue.
- Amazon contributes the highest share of orders.

### Operations Insights

- 93% of orders were delivered successfully.
- Return and cancellation rates remain low.
